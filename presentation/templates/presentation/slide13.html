Managing Static Files

Don't serve static files from your app server!

python manage.py collectstatic
Amazon S3 + Cloudfront provides a CDN that is within everybody's price range, or use Apache, Nginx, or your favorite static file webserver.

local_Settings.py: 
STATIC_URL = 'https://my-cdn.s3.amazonaws.com/'
